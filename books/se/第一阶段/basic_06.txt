1.方法
	1.1 什么是方法？为什么要使用方法？
		方法也就是函数(过程)，是指用于封装一段特定的逻辑功能，如执行计算的功能，并且可以在程序中反复被
	调用到的代码。使用方法可以减少代码的重复，便于程序的维护。
	1.2 方法的定义
		a)方法定义的五要素：修饰词，返回值类型，方法名，参数列表，方法体;
		如：
			public static int sum(int num1,int num2){
				return num1+num2;
			}
		其中：方法的参数是指：在调用时传递给方法，需要被方法处理的数据
				                    方法可有参数也可以没有参数，有参数可以使方法更加灵活。
		b)在方法定义时，需要声明该方法所需要的参数变量
		c)在方法调用时，会将实际的参数值传递给方法的参数变量，必须保证传递的参数类型和个数符合方法的声明
		d)方法调用结束时，可以返回一个数据，称之为返回值
		e)方法声明时，必须指定返回值类型
			-若方法不需要返回数据，将返回值类型声明为void
			-若方法需要返回值，将返回值类型声明为特定的数据类型
		f)return关键字
			-对于有返回值的方法，return有两个作用
				i)结束方法的调用
				ii)向调用方返回一个数据
				iii)return num1+num2;
			-对于没有返回值的方法，return只有一个作用
				i)结束方法的调用
				ii)returu;
	1.3 方法的调用
		a)方法的定义：
			public static int sum(int num1,int num2){
				return num1+num2;
			}
		  调用：
		  	i)
		  		int add = sum(3,5);
		  	ii)
		  		int a=23,b=45;
		  		int add = sum(a,b);
		b)堆栈内存结构，图略
			public static int max(int a,int b){}
			public static void main(String[] args){
				int a = 4;
				int b = 5;
				int myMax = max(a,b);
			}

		随机生成5个不重复的大写字母，然后用户从控制台输入5个大写字母，判断用户输入的字母个数对了几个，
