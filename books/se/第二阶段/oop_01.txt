1.对象的概念
	1.1 万物皆对象，客观存在的事物都是对象
2.类	
	2.1 什么是类？
		类定义了一种抽象数据类型	
		类就是模子，确定对象将会拥有的特征(属性)和行为(方法)
	2.2 类的特点 
		-类是对象的类型
		-具有相同属性和方法的一组对象的集合
3.什么是对象的属性
	属性-对象具有的各种特征
	每个对象的每个属性都拥有特定值
	例如：手机类
		属性：屏幕尺寸，CPU，内存。。。
	注：对象的属性拥有默认值，这个默认值与相应的类型对应
		例如：int默认值为0，boolean默认值为false，引用类型默认值为null
4.什么是对象的方法
	方法-对象执行的操作
5.类与对象的关系
	类是抽象的概念，仅仅是模板，在现实世界中是不存在的，而对象是一个你能够看得到，摸得着的具体实例
	例如：
		手机类：
			属性：屏幕，CPU，内存
			方法：打电话，发短信
		对象：Iphone5，Iphone4s，华为荣耀，三星Note等。。
6.如何定义类
	6.1 类的重要性：
		所有的java程序都是以类class为组织单元
	6.2 什么是类
		类是模子，确定对象将会拥有的属性和方法
	6.3 类的组成
		属性和方法
	6.4 定义类的步骤
		a)定义类名
			public class 类名{
			}
		class：定义类的关键字
		public：修饰词，后期详细讲解	
		b)编写类的属性(可以有属性，也可以没有属性)
			public class 类名{
				属性1的类型  属性1;
				属性2的类型  属性2;
				...
			}	
		c)编写类的方法(可以有方法，也可以没有方法)	
			public class 类名{
				属性1的类型  属性1;
				属性2的类型  属性2;
				...
				方法1;
				方法2;
				...
			}

7.如何定义java对象以及java中的引用变量
	7.1 使用对象的步骤
		a)创建对象(创建对象的步骤称之为实例化)
			类名 对象名 = new 类名();
			Telphone phone = new Telphone();
		b)使用对象	(通过"."来访问对象的属性和方法)
			i)引用对象的属性：
				对象名.属性 = 值;
				phone.screen = 5;
			ii)引用对象的方法
				对象名.方法名();
				phone.sendMessage();
	7.2 java中的引用变量
		a)引用类型的变量(简称引用)是为了能够对实例化的对象进行访问控制的，它内部存放的是该对象的
		     地址信息，当一个引用变量指向该类的对象时，就可以通过这个变量对这个对象实施访问
		b)通过"."操作来访问对象的属性和方法
		c)引用变量的赋值
			引用变量存储的是该对象的地址值，相同类型的引用类型变量可以相互赋值
			引用类型变量之间的赋值不会产生新的对象，但有可能使两个以上的引用指向同一个对象
			例如：(此处需要画堆栈内存图)
				Emp e1 = new Emp();//e1中存放的是创建出来的对象的地址值
				Emp e2 = e1;//将e1的值赋值给e2，所以e2中存放的也是刚刚创建出来的对象的地址值
				//换句话说，e1和e2指向同一个对象
				e1.name = "张三";
				e2.name = "李四";
				System.out.println(e1.name);//李四
8.Null和NullPointerException
	8.1 Null
		对于引用类型的变量，可以将其值赋值为null,null的含义为"空"，表示没有指向任何对象
		例如：
			Emp emp = null;
			emp = new Emp();//emp重新指向新创建的对象
	8.2 NullPointerException
		当一个引用的值为null时，我们不能通过这个引用来访问对象的属性和方法，否则会出现
		NullPointerException
		例如：
			Emp emp = null;
			emp.name = "张三";//空指针异常
